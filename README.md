## 📖 Overview

Complete **Spring Boot REST API** for **Product Management** with **MySQL** database and **HikariCP** connection pooling. No DTO validation - direct entity usage.

**Features:**
- ✅ Full CRUD operations
- ✅ MySQL + HikariCP connection pool
- ✅ JPA/Hibernate ORM
- ✅ Search by product name
- ✅ Production-ready configuration

## 🚀 Quick Start

### Prerequisites
```bash
# Required
- Java 17+
- MySQL 8.x Server
- Maven 3.8+


1. Clone Repository

git clone https://github.com/yourusername/aws_rds_product.git
cd product_aws_rds_db


2. Setup MySQL Database

-- Connect to MySQL as root
CREATE DATABASE product_aws_rds_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;



3. Configure Database
Edit src/main/resources/application.yml:

spring:
  datasource:
    url: jdbc:mysql://localhost:3306/product_aws_rds_db?createDatabaseIfNotExist=true&useSSL=false&serverTimezone=UTC&allowPublicKeyRetrieval=true
    username: root
    password: your_mysql_password  # ← UPDATE THIS


4. Run Application

mvn spring-boot:run


✅ Success! API available at: http://localhost:8080



📋 API Endpoints

| Method | Endpoint                         | Description        |
| ------ | -------------------------------- | ------------------ |
| POST   | /api/products                    | Create new product |
| GET    | /api/products                    | Read all products  |
| GET    | /api/products/{id}               | Read product by ID |
| GET    | /api/products/search?name={name} | Search by name     |
| PUT    | /api/products/{id}               | Update product     |
| DELETE | /api/products/{id}               | Delete product     |



🧪 API Examples (cURL)
1. Create Product

curl -X POST http://localhost:8080/api/products \
  -H "Content-Type: application/json" \
  -d '{
    "name": "iPhone 15 Pro",
    "description": "Latest Apple smartphone",
    "price": 999.99,
    "stock": 25
  }'



2. Get All Products

curl http://localhost:8080/api/products


3. Get Product by ID

curl http://localhost:8080/api/products/1


4. Search Products

curl "http://localhost:8080/api/products/search?name=iphone"


5. Update Product

curl -X PUT http://localhost:8080/api/products/1 \
  -H "Content-Type: application/json" \
  -d '{
    "name": "iPhone 15 Pro Max",
    "description": "Updated specs",
    "price": 1199.99,
    "stock": 15
  }'


6. Delete Product

curl -X DELETE http://localhost:8080/api/products/1



🏗️ Database Schema

-- Auto-generated by Hibernate
CREATE TABLE products (
  id BIGINT AUTO_INCREMENT PRIMARY KEY,
  name VARCHAR(255) NOT NULL,
  description VARCHAR(1000),
  price DOUBLE NOT NULL,
  stock INT
);
